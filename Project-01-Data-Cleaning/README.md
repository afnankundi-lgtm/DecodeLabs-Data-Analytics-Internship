# Project 01 – Data Cleaning & Preprocessing

## Overview

This project focuses on cleaning and preprocessing an e-commerce dataset using Microsoft Excel.

## Objectives

- Resolve missing values
- Verify duplicate records
- Standardize date formats
- Standardize text values
- Standardize numeric values

## Dataset

- Raw Dataset
- Cleaned Dataset

## Tools Used

- Microsoft Excel

## Techniques Applied

- Mode Imputation
- Duplicate Verification
- Date Standardization
- TRIM & PROPER
- ROUND Function

## Project Files

- Raw_Dataset.xlsx
- Cleaned_Dataset.xlsx
- Data_Cleaning_Report.pdf

## Result

A clean and standardized dataset was successfully prepared for further analysis.
